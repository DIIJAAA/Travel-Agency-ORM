✈️ Travel Agency Management System — ORM
Java application for managing a travel agency, built with Hibernate ORM and H2 database, developed as a collaborative team project as part of the DAM degree (Cross-Platform Application Development).

👥 Collaborative project — developed together with @HodaZaf and @Clara-dam

🛠️ Tech Stack

- Java 21 — core application logic
- Hibernate ORM 7 (Jakarta Persistence) — object-relational mapping
- H2 Database — embedded relational database (file-based)
- Maven — build and dependency management
- DAO Pattern — data access layer

🗂️ Features

- Client management (DNI, name, contact info)
- Booking system linked to clients and accommodations
- Accommodation and activity management
- Stay (Estada) tracking with composite keys
- Activity-booking relationship with composite keys
- Full CRUD operations via DAO classes
